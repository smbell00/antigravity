# Windsurf 사용자 매뉴얼

## 1. 개요 및 개발사
- **도구명**: Windsurf (윈드서프)
- **개발사**: Codeium (코디움)
- **한 줄 요약**: 개발자와 AI가 완전한 조화를 이루는 'Flow State(몰입 상태)'를 제공하기 위해 설계된 에이전틱 AI 통합 개발 환경입니다.
- **주요 키워드**: `#통합개발환경`, `#상용서비스`, `#VSCode기반`, `#에이전트루프`

---

## 2. 핵심 기능 및 AI 에이전트 기능
- **Cascade**: Windsurf의 핵심 에이전트 인터페이스입니다. 단순 채팅을 넘어 AI가 직접 터미널 명령을 제안/실행하고, 코드를 작성하며, 파일 구조를 분석하는 자율 에이전트 루프 역할을 수행합니다.
- **Supercomplete**: 타이핑 중에 실시간으로 다중 라인을 예측 완성해주며, 단순 문법 완성을 넘어 개발자의 의도에 맞게 대형 코드 블록을 제공합니다.
- **Flow State**: 개발자가 코드를 작성하는 흐름을 끊지 않고도 AI 에이전트에게 백그라운드 작업을 요청하거나 상호 피드백을 실시간으로 주고받을 수 있습니다.
- **Deep Codebase Awareness**: 프로젝트 내 파일들의 의존성을 깊이 분석하여, 특정 수정이 전체 코드에 미치는 영향도를 스스로 판단해 관련 파일을 함께 제안합니다.

---

## 3. 설치 및 설정 방법
1. [Windsurf 공식 웹사이트](https://codeium.com/windsurf)에서 설치 프로그램을 다운로드합니다.
2. 기존 VS Code 설정을 마이그레이션하여 원클릭으로 플러그인과 테마를 유지할 수 있습니다.
3. Codeium 계정으로 로그인한 뒤, Cascade 기능을 사용할 모델(Codeium의 자체 모델 또는 Claude 3.5 Sonnet 등)을 구성합니다.

---

## 4. 핵심 사용 명령어 및 워크플로우
- **Ctrl/Cmd + L**: Cascade 에이전트 창을 활성화하여 전체 프로젝트 대상 태스크를 지시합니다.
- **Cascade Autopilot Mode**: Cascade 대화에서 에이전트가 단계를 분석해 터미널 명령어 실행, 빌드 테스트 진행, 코드 수정을 연쇄적으로 처리하도록 수락 버튼을 클릭하여 흐름을 위임합니다.

---

## 5. 고급 팁 및 에이전틱 활용 방법
- **터미널 자동 실행 제안 활용**: Cascade가 에러 로그를 분석하여 터미널에서 패키지를 설치하거나 테스트를 실행할 때, 명령을 직접 복사할 필요 없이 에이전트 패널의 'Run' 버튼 클릭만으로 즉시 디버깅할 수 있습니다.
- **컨텍스트 멘션**: `#` 또는 `@` 기호를 사용하여 대화창에 특정 파일, 깃 브랜치, 폴더를 수동으로 연결해 타겟 분석 정밀도를 극대화할 수 있습니다.
