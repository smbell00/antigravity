# Cursor 사용자 매뉴얼

## 1. 개요 및 개발사
- **도구명**: Cursor (커서)
- **개발사**: Anysphere (아니스피어)
- **한 줄 요약**: VS Code를 포크(Fork)하여 AI 기능을 네이티브하게 통합한 차세대 AI 코드 에디터입니다.
- **주요 키워드**: `#통합개발환경`, `#상용서비스`, `#VSCode기반`

---

## 2. 핵심 기능 및 AI 에이전트 기능
- **Composer (Cmd + I / Ctrl + I)**: 여러 파일에 걸쳐 동시에 코드를 수정하고 새로운 기능을 구현할 수 있는 멀티 파일 에이전트 인터페이스입니다.
- **Tab (Auto-complete)**: 코드 작성 중 다음 입력을 예측하여 라인 및 블록 단위 코드를 자동 완성합니다.
- **Chat (Cmd + L / Ctrl + L)**: 현재 코드 컨텍스트나 프로젝트 전체 구조를 바탕으로 질문하고 답변을 받을 수 있는 코드 사이드바 대화 기능입니다.
- **Codebase Indexing**: 전체 프로젝트 코드를 벡터화하여 AI가 관련 파일을 쉽게 찾고 정확한 답변을 제안할 수 있도록 돕습니다.
- **Terminal CMD K**: 터미널에서 자연어로 명령어를 입력하면 적절한 CLI 명령어로 변환하고 실행해 줍니다.

---

## 3. 설치 및 설정 방법
1. [Cursor 공식 웹사이트](https://cursor.sh)에서 운영체제에 맞는 설치 파일을 다운로드합니다.
2. 기존 VS Code의 설정, 테마, 확장 프로그램을 원클릭으로 그대로 가져올 수 있습니다.
3. **API Key 설정**: Cursor Pro 요금제를 구독하거나 가입 시 제공되는 무료 크레딧을 사용하며, 필요한 경우 Settings -> Models에서 사용자 본인의 OpenAI, Anthropic 등의 API 키를 입력할 수 있습니다.

---

## 4. 핵심 사용 명령어 및 워크플로우
- **Ctrl/Cmd + K**: 파일 내 특정 코드 영역을 드래그한 후 질문하여 실시간 코드를 편집합니다.
- **Ctrl/Cmd + L**: 사이드바 Chat을 활성화하고 `@Files`, `@Folders`, `@Web`, `@Codebase` 등의 심볼을 사용하여 원하는 컨텍스트를 주입합니다.
- **Ctrl/Cmd + I (Composer)**: 복잡한 리팩토링이나 다중 파일 수정이 필요한 작업을 입력하여 자동으로 에이전트가 코드를 완성하고 적용(Apply)하도록 합니다.

---

## 5. 고급 팁 및 에이전틱 활용 방법
- **@Web 기능 활용**: 최신 라이브러리 공식 문서의 링크를 직접 제공하여 최신 API 스펙에 어긋나지 않는 고품질 코드를 생성하게 만듭니다.
- **Rules for AI (또는 `.cursorrules` 설정)**: 프로젝트 루트 폴더에 `.cursorrules` 파일을 작성하여 기술 스택, 코딩 스타일, 개발 원칙을 저장하면 AI가 항상 이를 엄격하게 준수합니다.
