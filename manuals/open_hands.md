# OpenHands 사용자 매뉴얼

## 1. 개요 및 개발사
- **도구명**: OpenHands (오픈핸즈 - 이전 OpenDevin)
- **개발사**: All-Hands AI 및 오픈소스 커뮤니티
- **한 줄 요약**: Devin의 오픈소스 대안을 지향하며, 사용자가 로컬 Docker 환경에서 안전하게 고도화된 자율 개발 에이전트를 구동할 수 있도록 돕는 웹 기반 에이전트 플랫폼입니다.
- **주요 키워드**: `#오픈소스`, `#에이전트루프`, `#터미널제어`, `#브라우저제어`

---

## 2. 핵심 기능 및 AI 에이전트 기능
- **Docker Sandbox Execution**: AI 에이전트가 실행하는 모든 코드와 빌드는 격리된 로컬 Docker 컨테이너 내에서 안전하게 구동되므로 사용자 호스트 시스템이 보호됩니다.
- **Micro-Agent Architecture**: 특정 소형 태스크를 처리하는 마이크로 에이전트들을 결합하여 유연하게 아키텍처를 교체할 수 있습니다.
- **Interactive UI Panel**: 파일 에디터, 가상 터미널, 브라우저 미리보기, 실시간 대화창이 올인원으로 통합된 미려한 반응형 웹 인터페이스를 제공합니다.
- **Multi-lingual Support & Large LLM Ecosystem**: 글로벌 커뮤니티 기반으로 전 세계 수많은 언어와 온프레미스/오픈소스 LLM API 백엔드를 완벽하게 지원합니다.

---

## 3. 설치 및 설정 방법
1. 로컬 환경에 **Docker**가 먼저 정상 설치되고 실행 중이어야 합니다.
2. 터미널에서 아래 Docker 명령어를 실행하여 서비스를 구동합니다:
   ```bash
   docker run -it \
     --pull always \
     -e SANDBOX_USER_ID=$(id -u) \
     -e WORKSPACE_BASE=$PWD \
     -v /var/run/docker.sock:/var/run/docker.sock \
     -v $PWD:/workspace \
     -p 3000:3000 \
     ghcr.io/all-hands-ai/openhands:latest
   ```
3. 브라우저로 `http://localhost:3000`에 접속하여 API 설정창에 본인의 LLM 키를 제공하고 실행합니다.

---

## 4. 핵심 사용 명령어 및 워크플로우
- **작업 개시**: 브라우저 화면 좌측 하단 대화란에 임무를 부여합니다.
- **워크스페이스 탐색**: 화면 중앙에 위치한 에디터 탭을 통해 AI가 수정한 소스코드를 실시간으로 열람합니다.
- **브라우저 프리뷰**: 에이전트가 연동 브라우저를 띄워 서버 렌더링을 시작하면, 화면 내부 브라우저 탭을 통해 결과를 육안으로 체크합니다.

---

## 5. 고급 팁 및 에이전틱 활용 방법
- **원격 서버 및 프록시 디버깅**: 복잡한 인프라가 얽힌 대규모 프로젝트는 Docker 볼륨 마운트 시 읽기 전용으로 설정하거나, 별도의 테스트 환경 호스트를 띄워 컨테이너 네트워크에 묶어놓으면 로컬 환경 훼손 우려 없이 극도로 효율적인 자율 테스트 디버깅이 가능합니다.
- **커스텀 에이전트 정의**: YAML 설정을 통해 에이전트의 내부 행동 지침이나 기법(Chain of Thought 가이드)을 정교하게 개조하여 커스텀 워크플로우를 주입할 수 있습니다.
