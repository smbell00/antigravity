# Mentat 사용자 매뉴얼

## 1. 개요 및 개발사
- **도구명**: Mentat (멘타트)
- **개발사**: Mentat 오픈소스 팀 및 기여자들
- **한 줄 요약**: 깃허브 리포지토리의 소스 코드 전체 컨텍스트를 이해하고, 개발자의 터미널 프롬프트 지시에 따라 다중 파일 편집과 디버깅을 고속 처리하는 텍스트 기반 CLI AI 코딩 비서입니다.
- **주요 키워드**: `#터미널도구`, `#오픈소스`, `#터미널제어`

---

## 2. 핵심 기능 및 AI 에이전트 기능
- **Direct File Editing**: 번거로운 포트 포워딩이나 샌드박스 설정 없이 로컬 디스크에 보관된 프로젝트 파일을 직접 일괄 편집하여 연산 속도를 보장합니다.
- **Interactive Multi-File Context**: 실행 시 수동 또는 자동으로 프로젝트 관련 파일을 인덱스에 태그하여 LLM이 관련 파일을 유기적으로 엮어 이해하도록 돕습니다.
- **Git Diff Preview & Revert**: 소스코드가 변경된 직후 CLI 화면에 아름다운 칼라 Git Diff 형태로 제안 사항을 띄워주고, 이를 엔터 키 입력으로 승인하거나 수락하지 않고 뒤로 되돌릴(Revert) 수 있는 편의성을 제공합니다.
- **Universal Code Parser**: 파이썬, 자바스크립트, 러스트, C++ 등 언어 종류를 가리지 않고 코드베이스 파일 구조 파서가 작동하여 구조적 편집을 돕습니다.

---

## 3. 설치 및 설정 방법
1. Python 3.10 이상이 설치된 터미널 환경에서 `pip` 패키지 관리자로 간편하게 내려받습니다:
   ```bash
   pip install mentat
   ```
2. 에이전트가 호출할 기본 API 키(예: OpenAI API Key)를 환경 변수에 심어줍니다:
   ```bash
   export OPENAI_API_KEY=your-api-key-here
   ```
3. 코딩하려는 프로젝트 폴더 최상위로 이동한 다음 `mentat` 명령을 입력하여 구동시킵니다.

---

## 4. 핵심 사용 명령어 및 워크플로우
- **`mentat <files/directories>`**: 작업하고 싶은 파일이나 폴더 경로를 나열하여 실행합니다. (예: `mentat src/ api/`)
- **컨텍스트 대화**: 구동된 대화형 프롬프트 상에 해결하려는 과업을 적습니다.
- **Diff 수락**: 변경이 필요한 경우, 멘타트가 코드를 가공해 화면에 컬러 Diff 창을 띄웁니다. 마음에 들면 `y`, 거절하려면 `n`을 눌러 흐름을 통제합니다.

---

## 5. 고급 팁 및 에이전틱 활용 방법
- **자동 파일 탐색 기능**: 굳이 처음에 특정 파일을 지정하지 않고 빈 명령어로 `mentat`만 치고 시작하더라도, 자연어 요청 내용에 담긴 키워드나 함수명을 스스로 인지해 연관도가 높은 파일을 자동으로 가져와 컨텍스트 목록에 자율 충당합니다.
- **`.mentat_config.json` 커스텀**: 해당 프로젝트 전용 설정 프로필을 작성하여 자주 쓰는 시스템 프롬프트 가이드라인(예: "모든 코드는 테스트 케이스를 동반할 것")을 적어놓으면, 에이전트가 항상 코드를 고칠 때마다 해당 가이드라인을 반영하도록 제어할 수 있습니다.
