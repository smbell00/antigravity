좌측 네트워크 그래프에서 원하는 코딩 에이전트 노드(파란색)를 클릭하면, 실시간으로 파싱된 최신 사용자 매뉴얼을 열람할 수 있습니다.

팁: 키워드 노드(분홍색)를 클릭하면 해당 키워드를 공유하는 에이전트들만 강조 표시됩니다.

로컬 파일 보안 경고 (CORS)

브라우저가 보안 정책(CORS)으로 인해 로컬 Markdown 파일을 직접 fetch()하지 못하고 있습니다.

해결 방법:

  1. 터미널에서 이 프로젝트 폴더로 이동합니다.
  2. 다음 명령어를 입력하여 로컬 웹 서버를 구동합니다: 
    npm run dev  또는  npx serve
  3. 터미널에 표시된 로컬 서버 주소(예: http://localhost:8080)로 다시 접속합니다.

우선은 내장된 백업 데이터로 매뉴얼 기능을 실행해 둡니다. (그래프 동작 및 설명서 읽기 가능)
