# Antigravity 사용자 매뉴얼

## 1. 개요 및 개발사
- **도구명**: Antigravity (안티그래비티)
- **개발사**: Google DeepMind (구글 딥마인드)
- **한 줄 요약**: 개발자와 긴밀히 협력(Pair Programming)하며 고도의 작업 계획 수립, 백그라운드 태스크 실행, 그리고 브라우저 제어까지 스스로 처리할 수 있는 강력한 에이전트형 코딩 비서입니다.
- **주요 키워드**: `#상용서비스`, `#에이전트루프`, `#터미널제어`, `#브라우저제어`, `#딥마인드`

---

## 2. 핵심 기능 및 AI 에이전트 기능
- **Planning Mode & Execution Mode**:
  - **기획 모드(Planning Mode)**: 복잡한 코드 변경 전 `implementation_plan.md`를 생성하여 개발자에게 피드백과 명시적인 승인을 요청합니다.
  - **실행 모드(Execution Mode)**: 기획안 승인 후 `task.md` 할 일 목록을 실시간 업데이트하며 자율적으로 작업을 수행합니다.
- **Browser Subagent**: 브라우저를 직접 띄우고(Crawl/Interact) 돔(DOM) 요소 클릭, 입력, 캡처 및 화면 비디오 녹화를 수행하여 UI를 테스트하고 디버깅합니다.
- **Background Tasks (Async Command Execution)**: 터미널 명령어를 실행하고 해당 명령어 완료 시까지 다른 작업을 동시에 병렬 처리하며 완료 알림을 받습니다.
- **Multi-file replacement & Search (ripgrep)**: 대규모 프로젝트 내에서 특정 패턴을 고속으로 grep 검색하고 여러 파일의 비정형 위치를 정교한 패치 포맷으로 한 번에 안전하게 일괄 편집합니다.

---

## 3. 설치 및 설정 방법
1. Google DeepMind의 Antigravity 에이전트 환경(예: VS Code Ext 혹은 독립 개발 터널 환경)을 실행합니다.
2. 필요한 경우 프로젝트 최상위에 환경 설정 프로필을 설정하고, 실행 권한 수준(Terminal access, Sandbox permissions)을 구성합니다.
3. 로컬 워크스페이스에 자동으로 연동되어 작동을 시작합니다.

---

## 4. 핵심 사용 명령어 및 워크플로우
- **에이전트 활성화**: 사용자가 복잡한 요구사항(예: "새 기능을 테스트하고 에러를 디버깅해 줘")을 채팅창에 전송합니다.
- **계획 수립 피드백**: 생성된 `implementation_plan.md`를 통해 승인(Approve) 또는 수정 요청 피드백을 보냅니다.
- **비동기 작업 진행**: 긴 빌드나 테스트 프로세스는 에이전트가 백그라운드로 예약(`schedule` 등)하여 실행하고, 완료되면 개발자에게 알립니다.

---

## 5. 고급 팁 및 에이전틱 활용 방법
- **슬래시 명령어 활용**:
  - `/goal`: 장기 실행 작업(Extra thorough)을 트리거하여 완전 해결까지 멈추지 않고 스스로 시도합니다.
  - `/schedule`: 반복적인 빌드 테스트나 주기적인 상태 확인을 자동화하는 크론(Cron) 타이머를 예약합니다.
  - `/grill-me`: 에이전트가 기획에 필요한 질문들을 대화식으로 면접하며 완벽한 설계를 다집니다.
- **브라우저 테스팅 연계**: UI 레이아웃 이슈 발견 시 브라우저 서브에이전트를 구동하여 직접 실동작 테스트 및 시각적 피드백 검증을 시킵니다.
